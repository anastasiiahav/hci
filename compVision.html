Assignments Workshop 'Computer Vision'

During this workshop I've worked with Processsing. Processing is an open source programming language and environment for people who want to program interactive images and animations. Processing is used by students, artists, designers, scientists and hobbyists to learn and develop concepts. Processing is used at universities of applied sciences and universities to teach the fundamentals of programming in a visual context, and functions as a digital sketchbook and professional production tool.

Processing

Assignment 1

With Processing you can not just print bits of text in the concole window, but also produce the graphical output. You can draw an ellipse with ellipse(50, 50, 80, 100);. You can also change color, border and position of the figure.
For this assignment I have written a program that draws three ellipses at various locations, with various colors, widths and heights and with and without borders.

Assignment 2

Beside ellipses you van draw other shapes, for example lines. To draw a line you need 'line(30, 20, 85, 75);'. The first two parameters represent the x and y coordinates of the starting point, the last two parameters represent the x and y coordinates of the end point.
For this assignment I have written a program that draws four different shapes in various colors, and at various locations.
Here below you can view printscreens from aligned code and a screenshot of the result in the display window.

Assignment 3

To make your code more dynamic it can be useful to also use variables. A variable can be seen as a virtual box with a name that you can temporarily store a value on. We call the storage of values in a variable, the declaration of variables. In Processing you declare variables before you create the void setup. To create a variable you need to use 'int nameofvariable'.
For this assignment I have written a program that draws two different shapes, with various colors and make them interactive.
First I drew a circle where the color is related to the location of the mouse. After that i drew a rectangle with the size related to the x location of the mouse.
Here below you can view printscreens from aligned code and a screenshot of the result in the display window.
